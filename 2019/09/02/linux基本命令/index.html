Linux 基本命令

linux常用命令：

  • ls 列出该目录下的文件
  • pwd 显示当前所在目录
  • cd [目录名] 切换目录
  • touch [文件名] 更新文件修改时间，如果文件不存在就创建新的文件
  • mkdir 创建新的文件夹
  • rm [文件名] 删除文件
  • clear 清屏

一、文件和目录操作的常用命令

终端使用技巧：

1.使用 tab 键自动补全或提示 文件名、目录、命令
2.使用上下光标切换历史命令
3.不想输入命令时，使用ctrl+c终止命令
